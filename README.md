# IZZY-Assistente-de-Estudos
Site desenvolvido com HTML, CSS e JavaScript. O usuário informa sua dúvida e recebe links de sites confiáveis que levam diretamente às respostas.

---

## Descrição
IZZY é um **assistente de estudos online** desenvolvido apenas com **HTML, CSS e JavaScript**, sem frameworks ou APIs.  
O site permite que o usuário **registre sua dúvida** e receba **links de sites confiáveis**, direcionando diretamente para informações relacionadas ao assunto.

O projeto é **simples, rápido e totalmente gratuito**, sem cadastro, sem banco de dados e sem coleta de dados pessoais.  
Este projeto foi desenvolvido para mostrar que é possível criar algo **útil e funcional** sem depender de dezenas de frameworks ou APIs.

---

## Funcionalidades
- Criação de dúvidas com seleção de matéria e descrição do conteúdo.
- Geração de **links confiáveis** para pesquisa rápida.
- Armazenamento das dúvidas apenas **na sessão do navegador**.
- Interface simples e amigável.
- O link da Khan Academy aparece somente para **Matemática e Português**. A disponibilidade de resultados depende do próprio site.

---

## Tecnologias Utilizadas
- **HTML**: estrutura do site  
- **CSS**: estilo e layout  
- **JavaScript**: lógica de criação e gerenciamento das dúvidas  
- **Google Fonts**: fonte principal  
- **Bootstrap Icons**: apenas para ícones  

Nenhuma outra biblioteca ou framework foi utilizada.

---

## Privacidade e Segurança
- **Não há coleta de dados pessoais**.  
- O site é **educativo e gratuito**, sem fins lucrativos.  
- Todo o conteúdo fica **localmente no navegador**.

## Como usar
1. Clique em **“Criar nova dúvida”**.  
2. Selecione a matéria e descreva sua dúvida.  
3. Clique em **“Enviar”** para gerar links confiáveis relacionados.  
4. Salve a dúvida para consultar novamente quando quiser.
